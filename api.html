JavaScript API

The API will make you happy

UI.Animate

var Animate = new UI.Animate({
	$element: $('#my-element'),
	animationDuration: 300
});

Animate.play(animationMethod, onDone)
Will apply an animation method on the $element, that pointed when you construct UI.Animate()

Animation methods

fadeIn
fadeOut
appear
disappear

UI.ClickOutside

var ClickOutside = new UI.ClickOutside({
	$element: $('#my-element'),
	onClickOutside: function($target){}
})

ClickOutside.bind()
Activate the click outside listener

ClickOutside.unbind()
Deactivate the click outside listener

UI.Popup

var Popup = new UI.Popup({
	width: 500,
	animationDuration: 500,
	onShow: function(instance){},
	onHide: function(instance){}
});

Popup.show(title, content)
Shows the popup

Popup.hide()
Hides the popup

Popup.showMessage(type, timeout, text)
Shows text with background under the title (it can be error message for example)

Popup.hideMessage()
Hides the popup message

Popup.changeContent(html)
Change the popup body content

Popup.changeTitle(html)
Change the popup title content

Popup.setWaitingMode()
Shows waiting animation on the popup title

Popup.removeWaitingMode()
Removes the popup waiting animation
